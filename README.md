# 疲劳驾驶检测系统

#### 运行环境：
##### ·jetson nano   ·
·jetpack 4.6.3 

·使用自带的tensorrt 8.2.1.8

·python 3.6.15

·torch 1.10

·opencv-python 3.4.14

·numpy 1.19.4

#### ·预期功能：
检测司机眼睛和嘴巴的状态，计算疲劳程度，判读是否疲劳驾驶，并将司机的行车状态数据转化成图表直观显示，根据疲劳的状态程度来使用语音播报提醒司机或发送报警邮件。

#### 主要文件说明：
·camera_dection是目前主要使用的检测方法

·权重文件在weights文件夹中
